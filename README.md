# JSON ↔ CSV Converter

A React app for converting data between JSON and CSV. It supports pasting text directly, uploading files, validating input, and exporting the converted result.

## Features

- Convert JSON → CSV and CSV → JSON
- Live validation for both formats
- Upload .json or .csv files
- Export converted data as a downloadable file
- JSON output is automatically formatted for readability

## Installation and Running

```
git clone https://github.com/Ray-AS/json-to-csv.git
cd json-to-csv
npm install
npm run dev
```

## Usage

1. Paste JSON or CSV into the text area, or select a file to upload.
2. Click Convert to generate the output.
3. Download the result if needed.
